## 📺 YouTube Channels 

### For Angular & Frontend

[Angular (Official Channel)](https://www.youtube.com/@Angular): Perfect for the official framework keynotes, feature deep-dives, and updates directly from the Google team.

 [Deborah Kurata](https://www.youtube.com/@deborah_kurata): Deborah is legendary for breaking down complex architectural patterns (especially RxJS and Signals transition) into incredibly digestible beginner-to-intermediate concepts.

[Joshua Morony](https://www.youtube.com/@JoshuaMorony): Highly recommended for modern Angular. He focuses heavily on high-performance, signal-based architectures and clean reactive patterns.

## For .NET & Backend

[Code Opinion](https://www.youtube.com/@CodeOpinion)
Derek Comartin - great .NET and architecture guy. DDD, CQRS, Event-Sourcing.

[Nick Chapsas](https://www.youtube.com/@nickchapsas): Intermediate-to-advanced C# performance tricks, clean architecture, and keeping up with the latest minimal API optimizations in .NET 10. (note: sometimes too performance focused, and library reviews are a bit weak.)

[Amichai Mantinband](https://www.youtube.com/@amantinband): Brilliant tutorials on clean architecture, design patterns, and building robust backend systems with ASP.NET Core.

[dotNET (Official Channel)](https://www.youtube.com/@dotnet): Great for the On .NET Live shows, architectural discussions, and official Microsoft updates.
